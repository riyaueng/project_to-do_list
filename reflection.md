# Reflection: Einsatz von GitHub Copilot

In diesem Projekt habe ich GitHub Copilot für folgende Aufgaben und Fragestellungen verwendet:

- Fehlermeldungen analysiert und Lösungen vorgeschlagen
- Fehlerbehebung bei React-spezifischen Problemen (z. B. "impure function" bei Date.now())
- Erstellung und Anpassung der README.md
- Allgemeine Tipps zu Best Practices in React und TypeScript

GitHub Copilot hat mir geholfen, schneller Lösungen zu finden, typische Fehler zu vermeiden und Best Practices umzusetzen.
